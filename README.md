# Number Sorter

This project is a simple number sorting tool developed using JavaScript. It was created as part of a programming tutorial, with a few personal modifications for learning purposes.

## About the Project

The application allows the user to generate a set of random numbers within a defined range, ensuring that no numbers are repeated. The user can choose how many numbers to generate and define the minimum and maximum values.

## Features

* Random number generation without repetition
* User-defined quantity of numbers
* Customizable minimum and maximum range
* Input validation to prevent invalid operations
* Display of generated numbers on the screen
* Reset functionality to clear inputs and results

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla)

## Concepts Applied

* DOM manipulation
* Loops and arrays
* Functions and code reuse
* Input validation and conditional logic
